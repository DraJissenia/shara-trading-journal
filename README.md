Shara Trading Focus Journal
Un diario de trading profesional y moderno para traders de futuros.
Características
📊 Registro detallado de operaciones
📈 Estadísticas de rendimiento
🎯 Análisis de setups y timeframes
📱 Interfaz moderna y responsiva
🌙 Modo oscuro
📝 Notas y lecciones aprendidas
📸 Soporte para imágenes de trades
Uso
Accede a la aplicación en http://localhost:3000
Registra tus operaciones con todos los detalles
Analiza tu rendimiento en las diferentes secciones
Aprende de tus operaciones y mejora tu trading
Contribuir
Las contribuciones son bienvenidas. Por favor, abre un issue para discutir los cambios que te gustaría hacer.
